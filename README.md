# Intelligent Enterprise Documentation Assistant

AI-powered multi-agent platform for enterprise technical documentation retrieval, release intelligence, API analysis, and troubleshooting assistance.

## Features

- Multi-agent orchestration
- Retrieval-Augmented Generation (RAG)
- MCP web search integration
- Release impact analysis
- Technical document indexing
- Source attribution & citations
- Observability & tracing

## Technology Stack

- Angular
- ASP.NET Core 8
- FastAPI
- LangGraph
- Qdrant
- Ollama
